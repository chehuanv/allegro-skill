# Allegro SKILL Toolbox

Allegro PCB Editor 自动化脚本工具箱，整合 EDA365、FANYI 等社区优质 SKILL 脚本。

## 功能模块

- **布局 (Placement)**: 按原理图分页放置、自动平衡布局、单层/多层切换
- **显示 (Display)**: 颜色设置
- **检查 (Check)**: 位号交叉检查、无位号检查、文字重叠检查、RefDes 调整
- **文本 (Text)**: 批量修改 RefDes、AutoSilk 工具集、文本居中、装配图位号
- **过孔 (Via)**: 替换/更新过孔、修改过孔网络
- **焊盘 (Pad)**: 创建焊盘
- **转换 (Conversion)**: BRD 转 PADS
- **输出 (Output)**: Film 重排序、CAM 输出/打包、一键导出 Gerber
- **搜索 (Search)**: 器件/网络搜索

## 安装

1. 下载本仓库所有文件
2. 将 `allegro.ilinit` 复制到 Allegro 的 pcbenv 目录
3. 重启 Allegro PCB Editor
